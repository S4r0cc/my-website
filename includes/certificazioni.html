Ott 2024 | Boolean Coding Week

Descrizione:

La Boolean Coding Week è una serie di incontri online dedicati all'apprendimento delle basi di HTML, CSS e JavaScript. Durante le sessioni, vengono utilizzate API per creare siti web dinamici, integrando funzionalità intelligenti basate sull'intelligenza artificiale.


Competenze acquisite:

  • HTML
  • CSS
  • JavaScript
  • PHP
  • Script in formato HTML
  • Intelligenza artificiale

Giu 2022 | Eipass Standard

Descrizione:

L’EIPASS (European Informatics Passport) è una certificazione composta da sette moduli, ciascuno dedicato a un ambito specifico delle competenze digitali. I moduli trattano i fondamenti dell’ICT, la navigazione e la ricerca di informazioni online, la comunicazione e collaborazione in rete, la sicurezza informatica (IT Security), l’elaborazione di testi, l’utilizzo di fogli di calcolo e la realizzazione di presentazioni. La certificazione attesta un livello di competenza intermedio.


Competenze acquisite:

  • Fogli di calcolo
  • Microsofot Excel
  • Microsofot Word
  • Microsofot PowerPoint
  • Problem solving
